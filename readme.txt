Displays information for candidates in elections set to be administered through the platform http://candideit.org/

== Installation ==

1. Upload to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add your Name and API key, if you dont have one, visit http://candideit.org/
4. Choose your Election to display
5. Relax and Enjoy :)
